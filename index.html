1-Qadam: Backend sozlamasi

Djangoning `ListView` klasi biz uchun hamma og'ir ishni bajaradi.

views.py
from django.views.generic import ListView
from .models import Portfolio

class PortfolioView(ListView):
template_name = 'portfolio.html'
context_object_name = 'portfolios'
paginate_by = 3 # Sehr shu yerda yuz beradi 🪄
queryset = Portfolio.objects.all()

2 Mustaqil Loyiha & Deploy

  • Mana shu PDP Junior loyihasini serverga deploy qilamiz.
  • Tayyor loyihani PythonAnywhere cloud servisiga bir marta muvaffaqiyatli yuklang (Deploy qiling).
